## Overview
This repository contains the tasks completed as part of the QSkill Internship under the Python Development domain. Out of the available tasks in Slab 1, the following 3 tasks were selected and completed.

## Tasks Completed

### Task 1: Data Analysis using Pandas and Matplotlib
**File:** `Data_Analysis_Pandas_Matplotlib.ipynb`

Loaded a dataset using Pandas and performed basic data analysis, including calculating averages and grouped aggregations. Created visualizations using Matplotlib and Seaborn, including:
- Bar charts (revenue by category, satisfaction by region)
- Scatter plots (units sold vs revenue, customer age vs satisfaction)
- Correlation heatmap

Insights were derived from the analysis and visualizations to understand patterns in the data.

### Task 2: House Price Prediction using Linear Regression
**File:** `House_Price_Prediction.ipynb`

Developed a Linear Regression model to predict house prices based on features such as size, number of bedrooms, bathrooms, age, and location. The workflow includes:
- Data preprocessing (one-hot encoding, train-test split, feature scaling)
- Model training using scikit-learn's LinearRegression
- Model evaluation using MSE, RMSE, MAE, and R-squared score
- Visualization of actual vs predicted prices, feature coefficients, and residuals

### Task 3: Matrix Operations Tool using NumPy
**File:** `matrix_operation_tool.ipynb`

Built an interactive Matrix Operations Tool using Python and NumPy with an ipywidgets-based interface. Users can input two matrices and perform:
- Addition
- Subtraction
- Multiplication
- Transpose
- Determinant calculation

Includes input validation and error handling for incompatible matrix dimensions and non-square matrices.
